Cómo usar

1
Instalá las dependencias: pip install flask flask-cors yt-dlp
2
Ejecutá el servidor: python server.py (debe estar corriendo en localhost:5000)
3
Abrí este archivo HTML en tu navegador, pegá la URL de YouTube y descargá.
4
El archivo se guarda en la carpeta downloads/ junto al servidor y tu navegador lo descarga automáticamente.
